# Proyecto OPTI_SJ — Grupo 2

## Cómo editar
- No modifiques `report.tex` salvo para incluir secciones.
- Escribe contenido en `sections/*.tex`.


## Bibliografía
- Archivo: `references.bib` (formato BibTeX).
- Cita en el texto con:
  ```tex
  \cite{clave}
  ```
- Para mostrar todas las referencias:
  ```tex
  
	nocite{*}
  	\printbibliography
  ```
